# NWK — 최종심사 PT 데모 스크립트

> 최종심사(10월) 15점 "발표" 배점 대비. 60초 실시연 + 5분 PT 슬라이드 흐름.
> 대사는 평서체 기본, 자연스러운 구어체로 읽기 좋게 작성.

---

## 🎬 60초 실시연 스크립트 (무대 위 실제 조작 동선)

**Scene 0 · 오프닝 (0~5초)**

> 화면 공유로 `https://nwkorea.com` 열어둠. 헤더 No Worries Korea 로고 + 영어 UI 상태.

대사: "외국인 관광객이 한국에서 가장 자주 묻는 한 가지 질문, '이 가격 맞나요?' — 저희 서비스가 3초 안에 답합니다."

**Scene 1 · 4개 언어 전환 (5~15초)**

> 헤더 우측 언어 스위처 → 한국어 → 일본어 → 중국어 → 다시 영어. 페이지 전체 즉시 재렌더.

대사: "한·영·일·중 4개 언어 전 화면에서 공공 관광데이터를 그대로 쓰고 있습니다. 한국관광공사 TourAPI 4개 서비스 병렬 연동이 기반입니다."

**Scene 2 · 바가지 판정 시연 (15~35초)**

> `/check` → 음료 카테고리 → 생수(500ml) → 5000원 입력 → 판정하기.

대사: "관광지 노점에서 생수 500ml를 5000원에 샀다고 가정합시다. 적정가 1000~2000원, 280% 초과 — 바가지 판정. 근거는 한국소비자원 참가격 공공데이터 링크로 바로 연결됩니다."

> 결과 카드의 "공식 근거" 링크 클릭 → `price.go.kr` 새 탭 (실제 이동).

대사: "모든 판정은 공공 출처와 직결. 저희가 만든 숫자가 아닙니다."

**Scene 3 · 익명 공유 → 홈 피드 (35~45초)**

> "익명으로 공유" 토글 켜진 채 제출. 홈으로 돌아가 히어로에서 방금 공유한 판정 카드 확인.

대사: "판정 결과는 이메일 첫 글자만 남기고 익명 공유됩니다. 다른 여행자들의 실시간 제보가 홈 피드에서 계속 쌓여갑니다."

**Scene 4 · 4언어 툴팁 + 부천 특화 (45~55초)**

> `/courses` → "Bucheon Nature Trail Half-Day" → 지도 뷰 → 핀 클릭 → 한국만화박물관 툴팁 (주소+설명 영어). 언어를 일본어로 전환 → 같은 핀 클릭 → 일본어 툴팁.

대사: "부천 테마 여행은 31개 코스, 13개 스타일, 5개 기간. TourAPI가 다국어 데이터를 제공하지 않는 부천 POI 38개는 저희가 직접 4개 언어 데이터셋을 구축했습니다."

**Scene 5 · 마무리 (55~60초)**

> 홈으로 돌아와 히어로 피드 한 번 더 강조.

대사: "공공데이터로 만드는, 걱정 없는 한국 여행. 접수 시점 기준 라이브 서비스입니다."

## 🧠 예상 질문 대비

**Q1. "82개 카탈로그 중 52개가 reference라는데, LIVE 아닌 건 왜 LIVE화 안 했나?"**
→ "대중교통·고궁 입장료·영화관람료는 정부·공공기관이 고시 형태로 공개하는 데이터입니다. 값 자체가 고정되어 LIVE 연동 필요성보다 공식 출처 링크가 심사 투명성에 더 적합합니다. 가변 데이터(외식·환율)는 LIVE 연동했습니다."

**Q2. "부천 POI 다국어 데이터셋을 직접 번역했다는데, 자동 번역과 어떻게 다른가?"**
→ "행정안전부 표준 로마자, 일·중 한자 표기 원칙에 맞게 수동 큐레이션했습니다. 또한 각 POI의 설명문은 개관 연도·위치·특색을 2~3문장으로 사실 기반으로 작성. 자동 번역에서 자주 나오는 지명·고유명 오역이 없습니다."

**Q3. "익명 판정 공유 데이터가 조작되거나 스팸될 수 있지 않나?"**
→ "로그인 사용자만 작성 가능하고, Firestore 규칙으로 카테고리·verdict·필드 화이트리스트 검증. 관리자 모더레이션 UI(`/admin/checks`)에서 숨김·삭제 가능. 악성 제보가 히어로에 노출되지 않도록 `hidden` 플래그로 즉시 차단합니다."

**Q4. "VWorld 대신 Kakao로 바꾼 이유?"**
→ "VWorld는 해외 IP에서 타일 차단 이슈가 있어 외국인 관광객 대상 서비스로 부적합. POI 데이터는 여전히 공공 TourAPI 그대로, 시각화 레이어만 Kakao로 전환해 해외 접근성을 확보했습니다."

**Q5. "향후 수익모델은?"**
→ "당분간 무료 공공 서비스로 운영. 12개월 이후 지자체 관광정책 피드백 데이터 판매 및 관광 사업자용 대시보드(유료) 두 축을 검토. 사용자 측은 계속 무료."
